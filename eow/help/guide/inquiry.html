2. 機能・サービスについて問い合わせる

「英辞郎 on the WEBお問い合わせフォーム」は、サービスや機能についてのご要望や、「英辞郎 on the WEB Pro / Pro Lite」へのログインなどに関するご相談をお送りいただくためのものです。

▼有料版「英辞郎 on the WEB Pro」のサブスクリプション（支払い）停止・解約については、こちら「ご利用の停止について」へ

▼アカウント（メールアドレスID）の削除をご希望の場合は、こちら「個人情報削除依頼」へ

お申し出にあたっては「個人情報の取り扱いについて」の「5 個人情報の開示等について」を事前にご確認ください。
  • お問い合わせをお送りいただく前に、「FAQ：よくある質問」のページもご覧ください。

(1) お問い合わせフォームを開く

アルクの「英辞郎 on the WEB 用お問い合わせフォーム」を開きます。

(2) ご利用のサービスを選ぶ

各サービスごとにお問い合わせ先が異なりますので、「ご利用のサービス」のプルダウンメニューから「英辞郎 on the WEB」「英辞郎 on the WEB Pro」「英辞郎 on the WEB Pro Lite」を選択してください。

(3) その他の必要事項とお問い合わせ内容を入力し送信する

お名前、メールアドレス、URL、お問い合わせ内容をご記入いただき、「個人情報の取り扱いについて」をお読みの上、「個人情報の取り扱いに同意する」にチェックしてから、「確認」ボタンを押してください。表示されたページで、ご記入いただいた内容をご確認の上、ページ下にある「投稿」ボタンを押してください。

  • 「英辞郎 on the WEB Pro」または「英辞郎 on the WEB Pro Lite」をご利用いただいているお客さまで、ログイン時に使用されているアドレス（ログインID）が回答メールの受信を希望されるアドレスと異なる場合は、ログインIDを「メールアドレスID」欄にご記入ください。
  • 「お問い合わせ内容」欄には、ご質問やご要望の内容をできるだけ具体的にご記入ください。
  • 収録データの修正・追加のリクエストについては、「収録データについて修正や追加を提案する」をご参照ください。
