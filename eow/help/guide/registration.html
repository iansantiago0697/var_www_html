1. ご契約内容の確認

「英辞郎 on the WEB Pro」にログイン後、検索窓の上にある「環境設定」をクリックするとページの最上部に「ご利用サービス」が表示されます。

ご利用中サービス確認画面


2. メールアドレスID・パスワードの変更

有料版「英辞郎 on the WEB Pro」はメールアドレスIDをご登録いただき、「英辞郎 on the WEB Pro」をご購入いただいた方がお使いいただけるサービスです。

メールアドレスIDとパスワードは、下記のページ（要ログイン）にて新しいものに更新していただけます。

>> メールアドレスの変更はこちら
>> パスワードの変更はこちら
現在のパスワードが分からない場合は、こちらの項目をご覧ください。

3. お支払い履歴の確認／カード情報の変更

「英辞郎 on the WEB Pro」アカウントページでは、ご注文のコースや、お支払い履歴・明細をご確認いただけます。
クレジットカード情報を変更されたい場合は、「サービス情報詳細」欄の「カード情報の変更・更新」ボタンから、ご利用カード情報の更新ページへお進みください。ご利用のカードが不明の場合は、「カード情報の照会」をクリックして、Stripe に照会してご確認いただけます。

4. アプリ内購入の場合

「英辞郎 on the WEB」アプリで「英辞郎 on the WEB Pro」をアプリ内購入された場合、上記アカウントページに情報は反映されません。

アプリ内課金によるご契約内容の確認・変更方法は、iPhone/iPad版の場合はApp Storeのサポートページ、Android版の場合はGoogle Playのサポートページにてご確認ください。
