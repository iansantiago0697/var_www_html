「Pro」のご利用停止・退会について

有料サービス「英辞郎 on the WEB Pro」のご利用停止（解約）方法と、その際の注意点をご案内します。

  • 1. 「Pro」ご利用停止に際してのご注意
  • 2. 「Pro」ご利用の停止方法
  • 3. 「Pro」ご利用の再開方法（サービス利用期間内）
  • 4. 「Pro」再注文の方法（サービス利用期間終了後）
※ユーザーアカウント（メールアドレスID）自体の恒久的な削除は、「個人情報削除依頼」フォームにて承っております。
こちらのリンクをクリックし、「個人情報お問い合わせ」の項目からご依頼ください。

1. 「Pro」ご利用停止に際してのご注意

ご利用停止のお手続きは、オンラインショップ「アカウントページ」から行っていただけます。お手続きに際して、以下の点にご注意ください。

  • 月額コースでご利用の場合は停止手続きを完了した月の月末まで、年額コースでご利用の場合は アカウントページの「サービス情報詳細」画面に記載のご利用期日まで、サービスをご利用いただけます（いずれの場合も、ご契約満了月の翌月１日からご利用いただけなくなります）。
  • いったん利用を停止されますと、単語帳、検索履歴などのデータや、カスタマイズした情報などは全て削除されます。
    ※誤って停止手続きをされた場合、ご利用期間が終了する前に再開の手続きをお取りいただくと、これらの情報が削除されることはありません。
  • 初月の利用料金無料は、オンラインショップでの初回のお申し込み時に限り適用されます。
  • 利用規約 および FAQ の「お支払い、登録情報変更、ご利用の停止などについて」もあわせてご参照ください。
  • スマートフォンアプリ内で定期購入された方は、各ストアにて定期購入のキャンセルを行ってください。

2. 「Pro」ご利用の停止方法

弊社オンラインショップで購入した有料版「英辞郎 on the WEB Pro」サービスの利用を停止されたい場合は、ご利用期日（月額コースの場合は月末）までに、「アカウントページ」の「サービス情報詳細」画面にて「停止する」ボタンをクリックしてください。
※上記操作でサービスの自動継続が停止されると、「サービス情報詳細」画面内の表示は、ご利用状況が「ご利用停止予定」に、「停止する」ボタンは「再開する」に変わります（次項・図２参照）。

3. 「Pro」ご利用の再開方法（サービス利用期間内）

前項のお手続き後、当月（年額コースの場合は「アカウントページ」に表示されたご利用停止日まで）の間に「英辞郎 on the WEB Pro」のご利用を再開されたい場合は、オンラインショップ「アカウントページ」の「サービス情報詳細」画面にて「再開する」ボタン（図２）をクリックすると、ご利用状況が「ご利用中」に変化し、次月以降も引き続きサービスをご利用いただけるようになります。
※お支払い情報の再入力は不要です。また、この期間内に再開手続きをいただいた場合、当月の利用料が2重に課金されることはありません。
