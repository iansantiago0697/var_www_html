「英辞郎 on the WEB Pro」お試し版 の基本的な検索についてくわしく説明します。

「英辞郎 on the WEB Pro」お試し版 では、検索条件に一致した見出し語の項目を EDP（Electronic Dictionary Project）制作の英和辞書データ『英辞郎』と略語辞書データ『略語郎』、例文データ『例辞郎』、和英辞書データ『和英辞郎』から検索できます。

1. 「英辞郎 on the WEB Pro」お試し版 のトップページ（または、検索結果ページ）に設置されたキーワード入力欄（下図１）に、検索したいキーワードを入力し「英和・和英」ボタン（下図２）をクリックする（Enterキーを押すだけでもOKです）
  • キーワードは、英語でも日本語でも構いません
    • cat と検索 ⇒ 見出し項目に「cat」が含まれる項目を英和データから検索
    • 国際 と検索 ⇒ 見出し項目に「国際」が含まれる項目を和英データから検索
  • 検索語の英字での入力は、大文字でも小文字でも同じ検索結果になります。また、全角でも半角でも同じ検索結果になります。大文字・小文字、全角・半角が混在していても構いません。
    • 以下の例は、すべて同じ検索結果が表示されます。
      cat / CAT / Cat / … ｃａｔ / ＣＡＴ / Ｃａｔ …
  • 検索語は、１単語だけでなく、２単語以上入力することもできます。
    複数の単語をスペースで区切って入力すると、それらの言葉のすべてが含まれる見出し語を検索できます（AND 検索）。
    • 「cat dog」※ と検索 ⇒「cat」と「dog」の両方が含まれる見出し語を検索
      ※ cat と dog がスペースで区切られている２単語の検索キーワード
    • 「経済 効果」※※ と検索 ⇒「経済」と「効果」の両方が含まれる見出し語を検索
      ※※ 経済と効果がスペースで区切られている２単語の検索キーワード
  • 複数の単語から構成される語彙（複合語・連語）をスペースで区切らずに検索すると、キーワードそのままの文字列を含む見出し語を検索します。
    • 「経済効果」※※※ と検索 ⇒「経済効果」（この検索キーワードどおり）を含む見出し語を検索
      ※※※ スペースでの区切りのない、ひとつながりの複合語
      「経済的効果」のように「経済」と「効果」の間に「的」など別の文字を含む見出し語は、「経済効果」では検索できません。「経済 効果」のようにスペースで区切ると、検索できます。
      （OR 検索や NOT 検索など、より高度な使い方は「高度な検索」をご参照ください）

関連情報（英辞郎 on the WEB）
【Memo】和英検索はどのように処理されているの？
  • 和英検索（日本語の見出し語を検索するとき）は、通常漢字で表記される単語は漢字で、カタカナで表記される単語はカタカナで、ひらがなで表記される単語はひらがなで検索すると、検索の効率が向上することがあります。
    • 「バラ と検索」⇒ 植物の名前としての「バラ」を含む見出し語などを検索
    • 「薔薇 と検索」⇒ 映画『薔薇の名前』など漢字で表記された見出し語などを検索
2.「英和・和英」ボタン（右上図２）をクリックする
  • 英和辞書と和英辞書のどちらを検索するかは、プログラムが自動判定します。
    検索キーワードの先頭の文字が英数字の場合は英和検索、先頭の文字が日本語の場合は和英検索、となります。
    ※ 英語と日本語の検索語が混在する場合の検索方法は、「高度な検索」の「AND 検索」の項目をご参照ください。
3. 検索結果が表示される

  • 検索条件に該当した見出し項目およびその訳語が上図のように表示されます
  • 検索結果の表示順は、概ね以下のルールで決定されます
    • 優先度高：完全一致
    • 優先度中：前方一致
    • 優先度低：部分一致（後方一致を含む）

関連情報（英辞郎 on the WEB）
【Memo】和英検索はどのように処理されているの？
  • 該当項目が50件を超えた場合、１ページめには先頭の50件めまでを表示します
    ※１ページごとの検索結果表示件数は環境設定にて変更が可能です
    50件め以降の該当項目については、ページ上部・下部にあるページ送り用のナビゲーションをご利用いただくと閲覧できます。
  • 検索結果には、検索条件に一致した見出し語、その訳語が表示されます
    見出し語によっては、発音記号や、品詞・分野ラベル、その見出し語を使った例文などが掲載されている場合があります。
    ※ 品詞や分野ラベルの凡例は、こちらをご参照ください。
    ※ 見出し語によっては、データに関連する情報として URL を表示するものがあります。URLはリンクになっていますので、クリックすることでリンク先の内容を参照できます（リンク先の URL の変更などの事情により、情報を表示できない場合もあります。あらかじめご了承ください）。
4. 続けて検索する
  • 検索キーワード入力欄に別の検索キーワードを入力し、「英和・和英」ボタンをクリックして続けて検索できます
    検索キーワード入力欄のユーザ設定が初期設定のままの場合、検索キーワード入力欄には前回の検索キーワードが残っていますが、それを削除して、新たな検索キーワードを入力してください。
    「クリア」ボタンをクリックして、前回の検索キーワードを消去することもできます。
