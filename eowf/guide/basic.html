基本的な検索

「英辞郎 on the WEB Pro」お試し版 では、検索条件に一致した見出し語の項目を EDP（Electronic Dictionary Project）制作の英和辞書データ『英辞郎』と略語辞書データ『略語郎』、例文データ『例辞郎』、和英辞書データ『和英辞郎』から検索できます。

1. 「英辞郎 on the WEB Pro」お試し版 のトップページ（または、検索結果ページ）に設置されたキーワード入力欄（右図１）に、検索したいキーワードを入力する
  • キーワードは、英語でも日本語でも構いません。

2.「英和・和英」ボタン（右上図２）をクリックする
  • 英和辞書と和英辞書のどちらを検索するかは、プログラムが自動判定します。
  • 「英和・和英」ボタンを押す代わりに、Enterキーを押すだけでもOKです。
3. 検索結果が表示される
  • 検索条件に該当した見出し項目およびその訳語が右図のように表示されます。
  • 該当項目が50件を越えた場合、１ページに50件ずつ（※）表示します。
    ※１ページごとの検索結果表示件数は環境設定にて変更が可能です。

4. 続けて検索する
  • 検索結果ページ上部にある検索キーワード入力欄に別のキーワードを入力し、「英和・和英」ボタンをクリック（もしくはEnterキーを押下）し、続けて検索できます。

条件を詳細にして検索（検索オプション）

検索キーワード入力欄の下にある「▼条件を詳細に指定して検索」というリンクをクリックすると、以下のような複雑な検索条件を設定することが可能になります。

1. すべての語句を含む（AND検索）
スペース区切りでキーワードを入力し、「この条件で検索」をクリックすると、入力したすべてのキーワードを含む見出し項目を検索できます。
※「単語間の語数を指定して検索」以外の項目は、一度に条件を指定して検索可能です。
  • 英語、日本語、英語と日本語の混在、すべて検索可能です。

2．フレーズを含む（フレーズ検索）
２語以上が連なるフレーズを入力し、「この条件で検索」をクリックすると、入力したとおり（文字列に完全一致）のフレーズを含む項目だけを検索することができます。

3. いずれかのキーワードを含む（OR検索）
スペース区切りでキーワードを入力し、「この条件で検索」をクリックすると、入力したキーワードの、いずれかを含む見出し項目を検索できます。

4. キーワードを含めない（NOT検索）
スペース区切りでキーワードを入力し、「この条件で検索」をクリックすると、入力したキーワードすべてのキーワードを含まない見出し項目を検索できます。
  • 英語、日本語、英語と日本語の混在、すべて検索可能です。

5. 変化形を含める（変化形検索）
「変化形を含める」にチェックを入れて検索すると、設定した条件に含まれる、動詞、名詞、形容詞の変化形も含めた見出し項目を検索できます。
  • 変化形とは、動詞の活用形、形容詞・副詞の比較級・最上級、名詞の単数形・複数形を指します。
  • 英語にのみ適用されます。

6. 単語間の語数を指定して検索
「単語１」「単語２」にキーワードを入力し、その間に挟まれた単語の数を指定して検索することができます。
  • 「単語１」「単語２」の欄には、複数の単語を入力できます。
  • 「単語１」「単語２」のどちらかが空欄になっていると検索できません。
  • 単語間の語数を「1 〜 5」のように、メニューから複数指定する場合、1 つめの数に、2 つめの数より大きな数字（「5 〜 1」など）を指定していると検索できません。
  • ほかの「条件を詳細に指定して検索」と同時に使用できません。

検索結果からの拡張検索

検索結果に表示されている見出し項目の右側についているのアイコンをクリックすると、その見出し項目で、整列、頻度集計、Wikipedia検索、Yahoo検索、Google検索をおこなうことができます。その見出し項目についてもっとくわしく知りたいときや、英単語やフレーズが実際にどのような英文で、どのぐらい使用されているかなどを調べたいときに便利です。

  • 整列と頻度集計は、英和検索時のみご利用いただけます。
    ※ 整列・頻度集計についての説明はこちら　
  • 長い例文では使用できません（アイコンが表示されません）。
  • 英和検索時は、英語版のWikipedia検索、Yahoo検索、Google検索をおこない、和英検索時はそれぞれ日本語版の検索をおこないます。
  • 2011/12/14 Google Scholar 検索をメニューに追加しました。
  • 2012/ 5/50 定義部（訳語部）でも拡張検索が可能になりました。定義部にて任意の範囲のテキストを選択していただくと、拡張検索のメニューが表示されます。不要な場合はメニュー最下部にある「表示をOFF」をクリックしていただくと、次回、ブラウザーを再起動されるときまで表示されません。

単語帳

検索結果に表示された見出し項目の左側についているのアイコンをクリックすると、その見出し項目を単語帳に登録することができます。

  • 各画面右上にある「単語帳」のリンクから単語帳に移動できます。
  • 単語帳は英和・和英それぞれ分けて登録されます。
  • 英和・和英あわせて5000件登録できます。
  • 登録できるのは100文字以内の見出し項目のみとなります（100文字を超える場合は、メッセージが表示されます）。
  • 検索結果ページの右側にある「単語帳登録履歴」に、最近登録した10件が表示されます。
  • 「単語帳登録履歴」の「>>すべてを見る」のリンクからも単語帳に移動できます。
  • 単語帳に登録した項目は、タグを付けて管理することができます。
  • 「選択した項目を印刷する」ボタンを押すと、チェックを入れた項目の印刷用ページが表示されます。
  • 「選択した項目を復習する」ボタンを押すと、チェックを入れた項目の復習画面が表示されます。
  • 見出し語に対して、200字以内でメモを設定することができます。

検索履歴

検索したキーワードの履歴の最新50件を保持します。

　
  • 各画面右上にある「検索履歴」のリンクから検索履歴に移動できます。
  • 英和、和英を問わず、検索した順に保持します。
  • 登録日時、見出し項目でソートが可能です。
  • 各項目の左端にあるグレーの「×」ボタンをクリックすると、その項目が削除されます。
  • 検索結果ページの右側にある「検索履歴」に、最近検索した10件が表示されます。
  • 「検索履歴」の「>>すべてを見る」のリンクからも検索履歴に移動できます。
  • 小窓モードでは、検索結果の一番下に最新の５件のみ表示されます。

小窓モード

多くの方からの「ウィンドウを小さくして使いたい」というリクエストにお応えした、シンプルな、ウィンドウサイズ可変表示モードです。ほかの作業の邪魔にならないように、ウィンドウを小さくして、画面の端などに開いておきたいときなどに便利です。

　
  • 各画面右上にある「小窓モード」のリンクをクリックすると、別ウィンドウで小窓モード表示を開きます。
  • 拡張検索機能はありません。
  • イディオムやフレーズのリンクは使用できません。
  • 検索履歴は、検索結果の一番下に最新５件のみ表示されます。
  • 単語帳登録は可能ですが、単語帳ページは小窓モード」に対応していませんので、通常モードでご覧ください。
  • 検索結果ページから小窓モードを開いた場合、通常モードの検索結果に表示している内容を引き継ぎます。
  • 小窓モードにある「通常モード」のリンクをクリックして通常モードに戻った場合、小窓モードで表示していた内容を引き継ぎます（通常モードがアクティブ状態となります）。

検索結果を共有する

「英辞郎 on the WEB Pro」お試し版 の検索結果ページを、Twitter、Facebook、Google+ などでシェアするためのURLを発行する機能です。発行されたURLで表示されるページは「英辞郎 on the WEB Pro」お試し版 ユーザー以外の方でも閲覧が可能です。

　
  • 画面右側の検索履歴表示部分の上にある「検索結果を共有する」というブルーのタブをクリックします。
  • タブが開き、検索結果共有用のURLが発行されます。
  • 「URLコピー」をクリックするとクリップボードにURLがコピーされ、タブが閉じます。
    ※ ctrl(command) + C でコピーすることも可能です。その場合、タブを閉じる際は画面のどこかをクリックしてください。
  • コピーしたURLをTwitter、Facebook、Google+ など、さまざまなサイトで共有できます。
  • 「英辞郎 on the WEB Pro」お試し版 のユーザーではない方でも閲覧可能ですが、検索や各種リンク先への移動はできません。
