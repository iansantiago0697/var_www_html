本キャンペーンは、2018年1月31日（水）で終了いたしました。

オンライン英和・和英データベース検索サービスの「英辞郎 on the WEB」。
ニュース記事を読んでいるときに単語をサッと調べたり、仕事で英文メールを書くときに例文をザーッと検索結果に並べて参照したりと、英語業務や学習に、とても便利にお使いいただいているウェブサービスです。

 

なかでも、「英辞郎 on the WEB Pro（有料版）」や「英辞郎 on the WEB Pro Lite （無料登録版）」を使いこなしている方にご好評をいただいているのが「単語帳」。単語だけではなく、フレーズも100文字まではまるごと登録でき、自分だけにカスタマイズしたオリジナル単語帳を作ることができます。

→これは便利！単語帳のかしこい使い方

 

このたび、「英辞郎 on the WEB Pro（有料版）」の「単語帳」の登録項目数上限が、5,000から20,000へと増量されました。「単語帳」増量記念として、「英辞郎 on the WEB Pro」を新たにはじめる方、現在ご利用中の方、皆さまに最大２回のチャンスがあるお年玉キャンペーンを実施中です。

 

※「英辞郎 on the WEB Pro Lite（無料登録版）」の「単語帳」は100項目まで登録できます。
