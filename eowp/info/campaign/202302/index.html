「英辞郎 on the WEB Pro」新規お申し込みキャンペーン！​

2023年2月1日（水）～2023年2月28日（火）の期間限定で、有料サービス「英辞郎 on the WEB Pro」が

【最大２カ月間無料】で利用できる新規お申し込みキャンペーンを開始します！

今すぐ申し込む >

キャンペーン概要​

キャンペーン適用期間中に、「英辞郎 on the WEB Pro」オンラインショップで「月額」「年額」コース、いずれかをお申し込みいただくと、月間基本料最大2カ月分が無料となります。

無料期間とご利用料金について​

「英辞郎 on the WEB Pro」お申し込み月の翌々月に、無料期間を追加します。申し込まれたコース、ご利用料金、無料期間は以下の通りです。

実施期間について​

2023年2月1日（水）～2023年2月28日（火）お申し込み分まで
※2023年2月28日（火）23：59（日本時間）までに「英辞郎 on the WEB Pro」ショップでのクレジットカード手続きを完了してください。

キャンペーン対象の方​

・キャンペーン適用期間中に、「英辞郎 on the WEB Pro」の「月額」「年額」コース、いずれかをお申し込みの方。
・アルクの「英辞郎 on the WEB Pro」ショップでお申し込みいただいた方が対象です。
※アプリ内課金（App Store／Google Playストア）や、法人様経由で「英辞郎 on the WEB Pro」をお申し込みいただいた場合は対象外です。

キャンペーン適用方法​

・キャンペーン適用期間中に、アルクの「英辞郎 on the WEB Pro」ショップで「英辞郎 on the WEB Pro」をお申し込みいただくと、自動的にキャンペーン適用となります。

注意事項​

・本キャンペーンの適用は、1ユーザーにつき1回限りです。
・キャンペーンの期間内に「英辞郎 on the WEB Pro」ショップから申し込む必要があります。
