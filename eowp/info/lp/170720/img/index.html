「英辞郎 on the WEB Pro」なら、英和例文数113万、英和見出し項目数204万、和英見出し項目数334万（2017年7月現在）が収録されています。

例えば、「provide」で英和検索すると、結果は19,113件（2017年7月現在）。

検索結果から使いたい語句や例文を探します。さらに、条件を指定して検索すれば、結果を絞り込むことができ、より速やかに目的の表現にたどり着けます。
また、手紙やビジネスメール特有の定型例文（レター文例）をまとめて表示することもできます。《前文》や《結語》のような、分類名を《　》に入れて検索すれば、その分類に含まれるレター文例をまとめて表示できます。

※例文のコピー＆ペーストは、ページトップ画面右上の「環境設定」>「定義部での拡張検索」を非表示(OFF)に設定して行ってください。

「英辞郎 on the WEB Pro」には、英和見出し項目数204万、和英見出し項目数334万が収録されています（2017年7月現在）。一般的な英和辞書（各5〜10万語）をはるかに超える圧倒的な収録数で、多分野にわたって専門的なデータがあります。

例えば、タンパク質について調べると、「タンパク質」（和英見出し）…2,625件、「protein」（英和見出し）…3,151件と、たくさんの検索結果が出てきます。

各専門分野の見出し項目が豊富なので、経営、経理、企画・マーケティング職から、法務、技術の専門職、さらには翻訳者、医療関係者にいたるまで、英語を使う多種多様な職種のプロフェッショナルたちに重宝されています。

「英辞郎 on the WEB Pro」なら、調べたい語が前後にどのような単語を伴って使われるかを調べることができます。

例えば、ある英単語の、文中でのつなぎ方を頻度別に調べるなら…「頻度集計」、文脈での使われ方を調べるなら…「整列表示」が大活躍します。

豊富なデータベースより、ターゲットとなる語彙の結果を抽出、自然な英語を、自信を持って書くことができます。

検索は単語だけではなく、フレーズやイディオムなど、長めのものもOK。それを含んだ例文が複数表示されます。

たくさんの例文を、ざーっとスクロールして見ることができるので、伝えたいニュアンスに近い文を探したり、使い方を知ることができます。

「英辞郎 on the WEB Pro」なら、検索履歴も自動で保有されます。また、検索中に気になった単語やフレーズは、右クリックで5,000項目まで「単語帳」に保存できます。タグをつけて整理することもできるので、自分専用の単語帳へとカスタマイズして使いたい時にサッと出せます。

「英辞郎 on the WEB Pro」は、パソコンブラウザだけではなく、スマートフォン（ブラウザ）、スマートフォン・タブレットアプリ（iOS、Android）でご利用いただけます。

ご利用中の「Pro」のメールアドレスIDとパスワードでログインすれば、追加料金なしで、検索履歴や単語帳をPCとスマホで同期させて使うことができます。資料作成などのデスクワークはパソコンで、移動中はアプリで知らない単語を発音させて調べたり、単語整理をしたりと、スキマ時間も有効利用できます。
※アプリでは、整列表示、頻度集計は使用できません。
