英和・和英見出し　各200万項目以上

スマートフォン（iPhone・Android）・iPadに最適化した、
ワンタップですぐに英和・和英検索ができるアプリ「英辞郎 on the WEB」。
PCブラウザー版との単語帳や履歴の共有、全英文の音声再生など、
アプリ版のみの強化機能も搭載。移動中でも簡単にデータの検索・管理ができます。

アプリのダウンロード・インストールは無料！

調べたい言葉を、検索窓に入力するだけ。
英語でも日本語でも、英語と日本語が混じった組み合わせでも、単語でもフレーズでもOK。

「前方一致」に加え、「部分一致」にも対応。
検索上級者も納得の「AND検索」や「OR検索」など演算子を使った検索もできます。

「英辞郎 on the WEB Pro Lite」にログイン*して検索すると、検索結果には文例も表示されます。

さらに、有料版「英辞郎 on the WEB Pro」にアップグレードすると、100万件以上の例文から検索。文例もたっぷり表示されます。

*ログインにはメールアドレスID、パスワードのご登録が必要です。

ログインすれば、検索した単語やフレーズを「単語帳」に登録できます。登録した単語はソートやタグで管理できるので、自分だけのオリジナル単語帳の作成もOK。英語業務のスピードアップ、英語学習にも役立ちます。


※「単語帳」は、メールアドレスIDとパスワードで「英辞郎 on the WEB Pro Lite」にログインしてご利用ください（100件まで登録可能）。有料版「英辞郎 on the WEB Pro」にアップグレードすると、最大20,000件の見出し項目を登録することができます。

アプリではすべての見出しがワンタップで音声再生できるうえ、検索結果の英単語やフレーズ、英語の例文など、どこでもお好きな英語を範囲選択すれば、合成音声で読み上げてくれます。

iPadなら、「Slide Over」と「Split View」機能を使って「英辞郎 on the WEB」アプリを画面に常駐させることができます。

※2017年1月現在、「Slide Over」はiPad Pro、iPad Air以降、iPad mini2以降でお使いいただけます。「Split View」はiPad Pro、iPad Air2、iPad mini4でお使いいただけます。

よくある質問

アプリの利用は無料ですか？

ダウンロード・インストール時点で課金されることはありません。
「英辞郎 on the WEB（無登録）」と「英辞郎 on the WEB Pro Lite（無料登録）」での検索は無料です（※通信費はお客様負担となります）。有料版「英辞郎 on the WEB Pro」のご利用に際しては、アプリ内課金もしくはアルクショップで別途、お申し込み・お支払いが必要です。

どの端末（機種）で使えますか？

「英辞郎 on the WEB」アプリは、スマートフォン（iPhoneおよびAndroid）、iPad、iPod touchの対応機種にインストールすることができます。最新の対応機種については、App Store、Google Playでご確認ください。
